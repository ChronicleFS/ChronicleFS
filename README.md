# ChronicleFS

A next-generation filesystem written in Rust, with tags, versioning, journaling, compression, and encryption.

## Rust 2024 Compatibility
This project targets **Rust 2024 edition**. Ensure your toolchain is up-to-date:

```bash
rustup update stable
rustup default stable
```
